Every blog has to start somewhere, and this is mine.

I've been meaning to start writing for a while now. Not because I have groundbreaking things to say, but because writing helps me think clearly. And if something I write ends up being useful to someone else — great, that's a bonus.

## What I Plan to Write About

Honestly, I'm not going to box myself in. This is a personal blog, which means I'll write about whatever I find interesting at a given moment. That could be:

- Something I learned recently
- A book, tool, or idea I found useful
- A project I'm working on
- Or just a random thought I want to get out of my head

## Why Hugo?

I chose Hugo because it's fast, free, and lets me write in plain Markdown. There's no database, no CMS, no monthly subscription. Just files. I like that simplicity.

Alright — that's enough for a first post. More soon.
